# 🌈 みんなの ゲームパック

保育園の「おしごと紹介」イベント用のゲーム集です。
**子どもたちが描いた絵や園の写真を取り込んで、そのまま遊べる** 14種類のミニゲームが入っています。

ブラウザだけで動きます(インストール・ビルド・ネット接続すべて不要)。

## 🎮 あそべるゲーム

| ゲーム | あそびかた | つかう素材 |
|---|---|---|
| 🧩 ぱずる | 1枚の絵がランダムな形のピースにバラバラに。組み立てよう(4〜20ピース) | 絵・写真なんでも |
| ❓ クイズ | 4択クイズ。問題は `js/quiz-data.js` で自由に編集 | — |
| 🌈 おえかき ぼうけん | 子どもの描いたキャラが手足を動かして歩く横スクロール。⭐を10こ集めてゴールへ | キャラの絵+背景の絵 |
| 🌳 みんなの せかい | 取り込んだ絵がぜんぶ画面の中でお散歩。タッチすると跳んで名前が出る(観客向け) | キャラの絵 |

はじめから「子どもの絵ふう」のサンプルキャラが入っているので、**何も取り込まなくても全ゲーム遊べます**。

### 🎤 キャラの声(あたらしい機能)

「とりこみ・せってい」でキャラごとに **4種類の声を録音**できます(1つ最長10秒)。

- **よろこび**(やったー! わーい) … 対戦で**勝ったとき**に鳴る
- **あいさつ**(こんにちは やっほー) … タッチしたときに鳴る
- **おどろき・いたみ**(いたっ! うわっ!) … タッチしたときに鳴る
- **しっぱい**(そんなぁ… ざんねん) … 対戦で**負けたとき**に鳴る

録音した声は次のゲームで自動的に使われます(未録音なら従来の効果音のまま)。

- **みんなの せかい**:キャラをタッチすると、よろこび・あいさつ・おどろき のどれかをランダムで再生
- **じゃんけん**:キャラが勝てば「よろこび」、負ければ「しっぱい」
- **よーいどん!(レース)**:声が重ならないよう、**勝ったキャラの「よろこび」だけ**を再生

> 声は画像と同じく **その端末のブラウザ内(IndexedDB)だけ**に保存され、公開サイトには含まれません。
> マイクは **`http://localhost` かHTTPS** で開いたときだけ使えます(バックアップzipには声も含まれます)。

## 🚀 起動のしかた

Chrome(または Edge)推奨です。

```sh
./serve.sh        # Mac / Linux
serve.bat         # Windows(ダブルクリックでOK)
```

起動したら ブラウザで **http://localhost:8765** を開いてください。

> `index.html` のダブルクリックでもほぼ動きますが、
> 「こえでジャンプ」「さかさクイズ」「キャラの声の録音」のマイクと「カメラでとる」はセキュリティ上
> `http://localhost` で開いたときだけ使えます。スクリプト起動がおすすめです。

Python が無いPCの場合は、任意の静的サーバー(`npx serve` など)でも動きます。

## 🌐 オンラインで公開する(URLひとつで開けるようにする)

完全な静的サイトなので、無料ホスティングにそのまま置けます。

### 方法1: GitHub Pages(このリポジトリから自動デプロイ・おすすめ)

`.github/workflows/deploy-pages.yml` を同梱済みです。

1. このブランチを **main にマージ**する
2. (自動で有効化されない場合)リポジトリの **Settings → Pages → Source を「GitHub Actions」** にする
3. 以降、mainにpushするたびに自動で公開・更新されます
   → URL: **https://nuigurusho.github.io/hoiku/**

※ 無料プランのGitHub Pagesは**リポジトリがPublic**である必要があります(Privateのまま公開したい場合は方法2へ)。

### 方法2: Netlify / Cloudflare Pages / Vercel(リポジトリを公開したくない場合)

- **Netlify Drop**(https://app.netlify.com/drop): このフォルダをブラウザにドラッグ&ドロップするだけ。アカウント不要で即URL発行
- Cloudflare Pages / Vercel: GitHubのPrivateリポジトリを接続して自動デプロイ(ビルド設定は「なし」、出力ディレクトリ「/」)

### オンライン公開についての大事なこと

- **子どもの絵・写真はネットにアップロードされません。** 画像は「開いたその端末のブラウザ内(IndexedDB)」にだけ保存されます。逆にいうと、当日使う端末それぞれで取り込みが必要です
- HTTPSで公開されるため、`localhost` 起動と同じく **マイク(こえでジャンプ)とカメラ取り込みがそのまま使えます**(むしろ便利になります)
- クイズの問題文(`js/quiz-data.js`)はリポジトリ/サイトに含まれて公開されるので、個人名など載せないようにしてください

## 📷 子どもの絵の取り込みかた

1. メニュー右下の **「とりこみ・せってい」** を開く(おとなのひとよう)
2. カテゴリを選ぶ
   - **キャラクターの絵** … 動き回るゲームで使う(白い紙に濃くはっきり描いたもの◎)
   - **はいけいの絵** … ぼうけんの舞台になる
   - **写真・絵** … パズル・まちがいさがし用(園の写真もここ)
   - **ふくわらいパーツ** … ふくわらい用の顔の絵(キャラクターの絵とは別に管理できます)
   - **トリミング用の元素材** … 切り出したい大きな絵や写真を高画質のまま保存しておく置き場
3. 「ファイルをえらぶ」or「カメラでとる」で追加
4. キャラの絵は **「うごきせってい」** で 3本の線(くび・こし・まんなか)を絵に合わせる
   → 隣のプレビューで歩き出したらOK!
5. まちがいさがし用の写真は **「まちがいスポット」** で消したい場所をタップして保存
6. ふくわらい用の顔の絵は **「ふくわらいパーツ」** で 目・まゆげ・鼻・口を四角く囲んで保存
   → ふくわらいで、パーツが抜けた顔と切り出したパーツで遊べます(白い紙に顔を大きく描くと◎)
7. 各絵の **「⬇DL」** ボタンで、取り込んだ絵・写真を1枚ずつ画像ファイルとして書き出せます
8. **「トリミング用の元素材」** に置いた絵は、**「✂トリミング」** から何枚でも切り出して各カテゴリへ保存できます

画像はブラウザの中(IndexedDB)に保存されます。**当日使うPCで取り込んでください。**
白背景の透過は自動なので、明るい場所で影が入らないように撮るときれいに切り抜けます。

## 💾 設定のバックアップ・お引っ越し(zip)

取り込んだ絵や設定は端末ごとに保存されるので、**別のPCへ丸ごと移したいとき**は
「とりこみ・せってい」画面の **バックアップ** から zip で書き出せます。

1. 元のPCの **「とりこみ・せってい → 💾 バックアップ」** で **「⬇️ ぜんぶ エクスポート(zip)」**
   → `hoiku-settings-日付.zip` がダウンロードされます
2. 当日使うPCで同じ画面を開き、**「⬆️ zip から インポート」** でその zip を選ぶ
   → 絵・写真・うごきせってい・まちがいスポット・ふくわらいパーツ・オリジナルクイズ・ランキングが丸ごと復元されます

zip の中身は「画像ファイル(images/)+ 設定(manifest.json)」です。外部ライブラリ・ネット接続なしで
ブラウザだけで作成/展開します。**インポートすると、その端末の現在の内容は置きかえられます**(実行前に確認あり)。

## ❓ クイズ問題の編集

`js/quiz-data.js` をテキストエディタで開いて書き換えるだけです。
書式の説明もファイル内に書いてあります。園にまつわる問題(後半)は
本番前に必ず内容を差し替えてください。

1. **みんなのせかい** を大画面に映しておく(自分の絵を探して盛り上がる)
2. **クイズ**・**かげえクイズ** は全員参加(声で答えてもらう)

## 🛠 技術メモ

- 素の HTML + Canvas + JavaScript。外部ライブラリ・ネット接続不要(オフライン動作)
- 効果音は WebAudio でその場で合成(音声ファイルなし)
- キャラの声は MediaRecorder で録音し dataURL で IndexedDB に保存(バックアップzipにも同梱)
- さかさクイズの逆再生は Web Audio API(`decodeAudioData` → チャンネル配列を反転 → `AudioBufferSourceNode`)。録音データはゲーム内でのみ使用し保存しない
- キャラは「くび・こし・中心」の3線で頭・体・両足に分割し、ペープサート(紙人形)方式でアニメーション
- 画像・声は IndexedDB に保存。`とりこみ・せってい → ぜんぶ削除` でリセット
